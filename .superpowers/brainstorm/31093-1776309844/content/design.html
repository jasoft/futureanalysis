设计方案：动态日期与自动导航

针对 download_futures_account_status.sh 的优化设计

1. 动态日期计算 (方案 B)

Shell 逻辑
# 获取当前月份的前一个月 (macOS date 命令)
DEFAULT_END=$(date -v-1m +%Y-%m)

# 参数处理
START_MONTH="${1:-2024-09}"
END_MONTH="${2:-$DEFAULT_END}"

说明

  • 如果用户不提供第二个参数，自动计算为上个月。
  • 使用 date -v-1m 兼容 macOS。

2. 自动页面导航

JavaScript 注入逻辑
const TARGET_URL = "https://investorservice.cfmmc.com/...";
if (window.location.href !== TARGET_URL) {
  window.location.href = TARGET_URL;
  return "NAVIGATING";
}

说明

  • 在执行月份选择前，检查当前 URL。
  • 如果不匹配，直接跳转并返回状态。
  • Shell 脚本检测到 "NAVIGATING" 状态时，会等待页面加载完成。

3. 鲁棒性增强

  • 增加页面加载检测：在导航后等待特定元素（如 tradeDate 选择框）出现。
  • 优化输出文件名：文件名将反映动态计算后的日期范围。
